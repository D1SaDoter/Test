## Проект сделан простым и компактным, под задачу из тестового задания. При дальнейшем развитии я бы добавил EF-миграции и отдельный API.

Стек:
- ASP.NET Core MVC (.NET 9)
- Entity Framework Core
- SQLite

#  Как запустить проект:
```bash
cd Test/DeliveryOrders
```
Затем запустить проект:
```bash
dotnet run
```
Так же проект можно запустить напрямую через Visual Studio. БД создается автоматически при запуске.
